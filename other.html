This was one of the first projects I successfully made that included JavaScript. I thought it would be amusing to see what random names your could make by mashing an adjective and a noun together, so I created this. It uses a list of 50+ adjectives and a list of 50+ nouns, and the program takes one of each and puts it together, creating a completely unexpected name.


This was the second project I created that included JavaScript. I wondered how long it would take to master something based on Malcom Gladwell's theory, so I created this engine, which calculates the time it would take one to master something based on the number of hours they practice a day and the number of days they practice a week.
